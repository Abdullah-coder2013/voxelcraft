## Installation

Run `pip install -r requirements.txt` and then run `main.py`.

# Game Controls

WASD - Move

1-4 - Change current blocks

Left mouse button - Break block

RIght mouse button - Place block

Shift - Crouch

Ctrl - Sprint

Space - Jump

E - Toggle inventory
